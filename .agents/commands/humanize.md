# humanize Codex Prompt

Use this prompt in Codex CLI, Codex App, or a Codex IDE session:

```text
humanize-korean skill을 사용해서 이 한글 글의 AI 티를 제거해줘.
모드: Fast
장르: 자동
강도: 기본

[윤문할 텍스트]
```

Fast mode uses the single-skill workflow. Add `모드: Strict` when you need separated detection, rewriting, fidelity audit, naturalness review, and final arbitration.
